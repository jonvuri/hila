C · Vector field
Each row carries vector lines in a left gutter — one per depth level it spans. Vectors point toward the last child at the next depth, with angle determined by distance (0 rows=0°, 1=35°, 2=50°, 3=58°, 4=63°). Own-depth strokes are prominent; parent strokes fade. An expanded version with 30+ rows exists to stress-test the pattern.

7 · Note face (locked-in wikilink)
Full note mockup with inverted title, property panel, wikilinks, and backlinks.
Design language
Created 2026-03-16
Status draft
Tags design ui tokens

The design language for hila draws on minimal structural elements — inverted block headings, corner notch containers, and a strict monochrome palette with violet accent. See Architecture notes for the underlying system design.

The outline face supports multiple visual themes, all rendering the same closure trait data. The rank trait determines ordering within each parent scope. Both traits are provisioned on demand per matrix.

What is a matrix in hila?
An independent typed SQLite table that serves as a data container. Its addressable units are rows.
B · Corner notch
What is a trait?
Per-matrix metadata tables (rank, closure) auto-provisioned on demand and shared by all consumers.
C · Inverted prompt
What is a face?
A view and interaction surface that plugins register. Face types declare slots that matrix columns bind to.
